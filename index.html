How it works
01

Copy the prompt

Click Copy prompt in the renderer. It gives you an editorial prompt that tells an LLM how to rewrite your source text as structured markdown with the right heading hierarchy.

02

Generate the markdown

Paste the prompt + your text into any LLM (Claude, GPT, etc.). The output uses # H1 as page separators and ## H2 as sections — no manual index.

03

Paste and render

Drop the markdown into the renderer, click Render →. You get a full 3-column doc with a navigable sidebar, right TOC, prev/next pagination, and dark mode.

04

Download the HTML

The rendered result is a self-contained .html file — no server needed. Share it, open it offline, or host it anywhere.

The prompt
Editorial rewrite prompt

This prompt instructs an LLM to rewrite long-form text as structured, compressed, navigable documentation — optimized for the renderer's # H1 page model. Paste your source text after the --- at the end.
